# HabitFlow - Ultimate Habit Tracker

![HabitFlow Banner](https://images.unsplash.com/photo-1484480974693-6ca0a78fb36b?auto=format&fit=crop&q=80&w=2072)
*(Concept Image)*

**HabitFlow** is a premium, minimalistic habit tracking application designed to help you build consistency and achieve your goals. Built with a focus on privacy and aesthetics, it runs entirely in your browser.

## üåü Features

*   **Premium Glassmorphism UI**: A stunning dark-mode interface with smooth animations.
*   **Privacy First**: All data is stored locally in your browser (`localStorage`). No servers, no sign-ups required.
*   **Multi-User Support**: Separate profiles on the same device.
*   **Analytics Dashboard**: Visualize your progress with interactive charts and consistency scores.
*   **Calendar View**: Track your daily streaks with a GitHub-style contribution graph.
*   **Goal Tracking**: Set monthly targets and track your progress percentage.

## üöÄ How to Run

Since HabitFlow is a static web application, you don't need to install anything!

1.  **Clone the Repo**
